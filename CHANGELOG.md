# Журнал изменений

## 2025-12-09

Приложение переведено из стадии бета-тестирования в режим полноценной эксплуатации.
Обновление включает большое количество исправлений и улучшений.

Ключевые изменения:

* Пользователи создают и редактируют камеры и группы в приложении (так же, как в проекте [Камеры](https://github.com/vladpen/cams)).
* Добавлен экспорт и импорт настроек камер и групп на стороне пользователя.
* Вход по паролю заменён на ограничение доступа по ключам камер. Добавлена защита от утечки ключей.
* Оповещение о движении в охранных зонах использует системные механизмы Notifications API и Server Sent Events.
* Ярлык приложения PWA может вести на любую камеру или группу.
Для локальной сети и интернета ярлыки выполнены визуально разными.

## 2024-07-08

* Локализация (en, ru)
* include() в HTML-шаблонах
* Фабрика рендеринга шаблонов

## 2024-01-19

* Миграция с HTTPServer на asyncio. Неблокирующее чтение файлов реализовано через asyncio.subprocess.PIPE
* Возможность работы по протоколу HTTP за прокси-сервером, указание SSL сертификата в файле конфигурации необязательно
* Рефакторинг


## 2023-06-30

Добавлена возможность работы с детектором движения камеры.

***Внимание!***
В файл конфигурации добавлены обязательные параметры с префиксом "events".
Также изменены имена параметров с префиксом "web" (подробнее: git log -1 -p server/config-exapmle.py).
